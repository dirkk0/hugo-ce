Oberheim Matrix 1000 Editor

2026/05/20
Technologies: JavaScript WebMIDI Audio AI

A browser-based, real-time patch editor for the classic Oberheim Matrix 1000 analog synthesizer — no installation, no backend.

Using the Web MIDI API, the editor speaks Oberheim’s SysEx protocol directly to the synth: all 100 parameters grouped by signal flow, the full 10-slot modulation matrix, patch read and store, and a live MIDI monitor. Move a slider and the change reaches the hardware in milliseconds.

It was built largely with AI assistance (Claude) as part of our exploration into AI-assisted engineering — see the news post for that story.
